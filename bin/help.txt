general notes: if no repository='' or app='' is specified git will consider the current app as the repository (app root folder)
also please note that for all remote commands git will use the user and passwort etc defined in git.conf in the git app local folder
commit - commits all changes, will silently do an add all
add - adds all silently unless paths='' are specified
init - initializes a git repo locally
ls - list files
status - status
treechanges
branchlist
fetch - from remote
createbranch
deletebranch
printcommit
log
show
help
